# Kokoro TTS — Frontend

React + Vite frontend for the Kokoro TTS API.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Configuration

Set `VITE_API_BASE_URL` in a `.env` file to override the default backend URL
(`http://<hostname>:8000`):

```env
VITE_API_BASE_URL=http://localhost:8000
```
